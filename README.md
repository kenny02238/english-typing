# 🎯 AI 英文聽寫練習系統

一個由 AI 驅動的英文聽寫練習工具，支援漸進式 chunk 練習、即時語音朗讀與智能糾錯。

## ✨ 功能特色

- 🤖 **AI 即時生題**：根據你的偏好動態生成練習題目
- 📝 **漸進式練習**：從小片段開始，逐步組成完整句子
- 🔊 **語音朗讀**：自動播放英文，支援調整語速
- ✅ **即時糾錯**：錯誤單字即時高亮顯示
- 🌏 **完整解釋**：顯示繁體中文翻譯與單字解釋

## 🚀 快速開始

### 1. 申請 Gemini API Key

前往 [Google AI Studio](https://makersuite.google.com/app/apikey) 申請免費 API Key

### 2. 設定環境變數

在專案根目錄建立 `.env.local` 檔案：

```bash
GEMINI_API_KEY=你的_Gemini_API_Key
```

### 3. 安裝並啟動

```bash
npm install
npm run dev
```

打開瀏覽器訪問 [http://localhost:3000](http://localhost:3000)

## 📖 使用方式

### Step 1: 選擇練習條件

在首頁選擇你的偏好：
- **主題**：旅遊、餐廳、商務、日常等（可多選）
- **句子長度**：短句、中句、長句
- **難度等級**：A1/A2/B1/B2/C1/C2/C3

### Step 2: 開始練習

1. 系統自動播放英文片段
2. 在輸入框中逐字輸入
3. 按 Enter 或點「提交」檢查答案
4. 錯誤會標紅並顯示正確答案
5. 全對後自動進入下一個片段

### Step 3: 查看結果

完成整句後會顯示：
- 完整句子的中文翻譯
- 每個單字的解釋
- 點「繼續練習」生成下一題

## 💡 注意事項

- 語音功能需使用 Chrome/Edge 等現代瀏覽器
- 免費 API 配額：每天 20 次請求
- 配額用完時會顯示提示訊息
